# Wireshark: The Basics

Wireshark to graficzny analizator protokołów sieciowych. Pozwala na inspekcję pakietów w czasie rzeczywistym.

### Czego się nauczyłem:
* **Filtrowanie:** Używanie filtrów (np. `http`, `ip.addr == x.x.x.x`), aby wyłowić interesujący ruch z tysięcy pakietów.
* **Kolorystyka:** Zrozumienie, że kolory pomagają szybko zidentyfikować błędy (np. czarny/czerwony dla błędów TCP).
* **Follow TCP Stream:** Technika składania fragmentów pakietów w czytelną rozmowę między klientem a serwerem.
